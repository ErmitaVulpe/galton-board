# Galton board simulation

Galton board simulation demonstrating how repeated random left/right movements produce a normal (Gaussian) distribution and visualizing the emergence of statistical patterns from randomness.
## Run Locally

This project has many dependancies, like the [cargo toolchain](https://rust-lang.org/tools/install/) since it's made in rust. All dependancies are listed in `flake.nix`, so if you have the nix package manager installed, you can just run `nix develop` in the project root.

Clone the project

```bash
https://github.com/ErmitaVulpe/galton-board.git

cd galton-board

# Build in release mode
just

# Build in debug mode
just dev
```
